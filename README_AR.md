# AlterEgo AI - التوأم الرقمي الخاص بك

تطبيق ويب متقدم لإنشاء نسخة ذكية من شخصيتك تساعدك في توليد محتوى وردود رسائل احترافية.

## الميزات الرئيسية 🌟

### 1. ملف شخصي ذكي
- اختر شخصيتك وصفاتك
- أضف اهتماماتك وعاداتك
- نسخ صوتك الرقمي (اختياري)

### 2. صانع محتوى ذكي
توليد محتوى احترافي:
- **تغريدات**: منشورات Twitter قصيرة وجذابة
- **ريلز**: سيناريوهات فيديو احترافية
- **بوستات**: منشورات وسائط اجتماعية طويلة

اختر من 4 صيغ مختلفة:
- أصيل (Authentic)
- ملهم (Inspirational)
- فكاهي (Humorous)
- احترافي (Professional)

### 3. مساعد دردشة دبلوماسي
احصل على ردود ذكية:
- الصق أي رسالة معقدة
- اختر أسلوب الرد
- احصل على اقتراحات متعددة
- استخدمها فوراً

أساليب الرد:
- 🤝 دبلوماسي: هادئ ومتوازن
- 💪 حازم: مباشر وصريح
- 😄 فكاهي: خفيف وممتع
- 🎩 رسمي: احترافي وكوردي

### 4. دعم اللغات
- ✅ English (LTR)
- ✅ العربية (RTL)
- تبديل فوري بين اللغات

### 5. إعدادات متقدمة
- ⚙️ إدارة مفتاح Gemini API
- 🔒 تخزين آمن في Supabase
- ✅ اختبار الاتصال
- 📊 معلومات الاستخدام

## البدء السريع ⚡

### الخطوة 1: فتح التطبيق
```
تطبيق AlterEgo AI مُتاح على الويب
لا يتطلب تثبيت - يعمل مباشرة من المتصفح
```

### الخطوة 2: ملء ملفك الشخصي (اختياري)
1. اذهب إلى تبويب **"الملف الشخصي"**
2. اختر شخصيتك وصفاتك
3. أضف اهتماماتك
4. انقر "حفظ الملف"

### الخطوة 3: استخدام الميزات
- **بدون API**: استخدم الأمثلة التجريبية
- **مع API**: احصل على محتوى ذكي حقيقي

### الخطوة 4: إضافة مفتاح Gemini (اختياري)
1. احصل على مفتاح من: https://makersuite.google.com/app/apikey
2. اذهب إلى تبويب **"الإعدادات"**
3. الصق المفتاح
4. اختبر الاتصال ✅
5. احفظ المفتاح

- **الواجهة الأمامية**: React Native + Expo
- **قاعدة البيانات**: Supabase
- **الذكاء الاصطناعي**: Google Gemini API
- **الاستضافة**: Web

### معمارية الملفات
```
project/
├── app/(tabs)/
│   ├── index.tsx (الملف الشخصي)
│   ├── content.tsx (صانع المحتوى)
│   ├── chat.tsx (مساعد الدردشة)
│   └── settings.tsx (الإعدادات)
```

## الأسئلة الشائعة ❓

### هل أحتاج مفتاح API لاستخدام التطبيق؟
**لا.** يمكنك استخدام التطبيق مع الأمثلة التجريبية. المفتاح اختياري لتوليد محتوى ذكي حقيقي.

### كيف أحصل على مفتاح Gemini API؟
1. اذهب إلى https://makersuite.google.com/app/apikey
2. انقر "Create API Key"
3. انسخ المفتاح والصقه في الإعدادات

### هل البيانات آمنة؟
**نعم.** كل البيانات محفوظة في Supabase بأمان مع سياسات RLS.

### هل يعمل على الهاتف؟
**نعم.** التطبيق مُصمم للويب ويعمل على جميع الأجهزة.

### ما اللغات المدعومة؟
حالياً: العربية والإنجليزية. يمكن إضافة لغات أخرى لاحقاً.

### هل هناك نسخة محمولة؟
التطبيق الحالي للويب فقط. يمكن إنشاء تطبيقات محمولة مستقلة لاحقاً.
