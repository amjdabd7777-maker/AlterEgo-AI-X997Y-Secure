# إعداد Gemini API - AlterEgo AI

## الملخص
تم دمج Gemini API بالكامل في التطبيق. يمكنك الآن توفير مفتاح API خاص بك لتفعيل الميزات الذكية الكاملة.

---

## الخطوات الأساسية

### 1. الحصول على مفتاح Gemini API

1. اذهب إلى **Google AI Studio**: https://makersuite.google.com/app/apikey
2. انقر على **"Create API Key"**
3. اختر أو أنشئ مشروع جديد
4. سيظهر لك المفتاح - **انسخه**

### 2. إدخال المفتاح في التطبيق

1. افتح تطبيق AlterEgo AI
2. انتقل إلى تبويب **"الإعدادات"** (الأخير في القائمة السفلية)
3. في قسم **"مفتاح Gemini API"**:
   - الصق المفتاح في حقل الإدخال
   - اضغط **"اختبار الاتصال"** للتحقق من صحة المفتاح
   - اضغط **"حفظ المفتاح"** لحفظه بأمان في Supabase

### 3. التحقق من الحالة

- إذا رأيت رسالة **"متصل!"** - المفتاح يعمل بشكل صحيح ✅
- إذا رأيت رسالة **"خطأ"** - تحقق من المفتاح وحاول مرة أخرى

---

## الميزات الممكنة بعد التكوين

بعد إضافة مفتاح API، ستتمكن من:

### تبويب المحتوى (Creator)
- **توليد التغريدات**: إنشاء تغريدات باستخدام Gemini بناءً على موضوع وصيغة محددة
- **سيناريوهات الريلز**: إنشاء سيناريوهات فيديو كاملة بصيغة احترافية
- **البوستات**: توليد منشورات وسائط اجتماعية طويلة

### تبويب المحادثة (Chat)
- **ردود ذكية**: الحصول على ردود مخصصة بناءً على:
  - نص الرسالة المستلمة
  - السياق (مثل: "هذا مديري")
  - أسلوب الرد المطلوب (دبلوماسي، حازم، فكاهي، رسمي)

---

## البيانات الشخصية

جميع المحتويات المُنشأة:
- يتم حفظها في قاعدة البيانات الخاصة بك بأمان
- لا تُشارك مع أطراف ثالثة
- مفتاحك محفوظ بتشفير في Supabase

---

## الحالات

### ✅ API مُكوَّن
- جميع الميزات الذكية **مفعّلة**
- المحتوى يتم توليده باستخدام Gemini
- البيانات تُحفظ في السجل

### ⚠️ API غير مُكوَّن
- الميزات الذكية **معطّلة**
- التطبيق يستخدم بيانات وهمية للمراجعة فقط
- يمكنك تصفح الواجهات وفهم الميزات

---

## استكشاف الأخطاء

### المفتاح غير صحيح
- تحقق من نسخ المفتاح بالكامل (بدون مسافات)
- تأكد من أنك استخدمت أحدث نسخة من المفتاح

### خطأ الاتصال
- تحقق من اتصالك بالإنترنت
- ربما تم تعطيل المفتاح - أنشئ واحداً جديداً

### مشكلة في الحفظ
- قد تحتاج إلى تسجيل الدخول بحساب Supabase أولاً
- تأكد من أن لديك الصلاحيات الكافية

---

## الأمان والخصوصية

⚠️ **تحذير مهم:**
- **لا تشارك مفتاح API الخاص بك** مع أحد
- **لا تنشره** على مستودعات Git العامة
- المفتاح يُخول أي شخص بالعمل مع حسابك في Google AI

✅ **الممارسات الأمنية:**
- يتم تخزين المفتاح محلياً في التطبيق فقط
- يتم حفظه في Supabase بشكل آمن
- يُمكنك تغييره أو حذفه في أي وقت
