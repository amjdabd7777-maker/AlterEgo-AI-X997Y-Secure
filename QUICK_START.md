# البدء السريع - Quick Start

## ما هو AlterEgo AI؟
تطبيق ذكي يحول كتابتك إلى محتوى احترافي وردود رسائل ذكية بأسلوبك الفريد.

---

## 5 خطوات للبدء

### 1️⃣ فتح التطبيق
```
افتح المتصفح واذهب إلى URL التطبيق
```

### 2️⃣ اختيار اللغة (اختياري)
```
انقر على زر اللغة في أعلى يمين أي شاشة
الافتراضي: الإنجليزية
```

### 3️⃣ ملء الملف الشخصي (اختياري)
```
اذهب للتبويب الأول "Profile"
- اختر الشخصية والصفات
- أضف الاهتمامات
- احفظ الملف
```

### 4️⃣ استخدام الميزات
```
✨ تبويب "Creator": توليد محتوى
💬 تبويب "Chat": ردود ذكية
⚙️ تبويب "Settings": إعدادات
```

### 5️⃣ إضافة مفتاح API (اختياري)
```
للحصول على توليد ذكي حقيقي:
1. احصل على مفتاح من Google AI Studio
2. اذهب للإعدادات
3. أدخل المفتاح
4. اختبر الاتصال
```

---

## الاستخدام بدون API
```
✅ تصفح الواجهات
✅ شاهد الأمثلة التجريبية
✅ افهم الميزات
❌ لا توليد ذكي حقيقي
```

## الاستخدام مع API
```
✅ توليد محتوى ذكي
✅ ردود مخصصة
✅ حفظ في السجل
✅ الكل بلغتك المفضلة
```

---

## الإعدادات المهمة ⚙️

### اللغة 🌐
```
زر اللغة في كل شاشة
- English → عربي
- العربية → English
```

### مفتاح API 🔑
```
اذهب: Settings → Gemini API Key
الخطوات:
1. احصل على المفتاح
2. الصقه هنا
3. اختبر الاتصال ✅
4. احفظ
```

### البيانات 🔒
```
محفوظة في Supabase
آمنة وخاصة
يمكنك الوصول في أي وقت
```

## الأخطاء السريعة 🐛

| المشكلة | الحل |
|--------|------|
| المفتاح لا يعمل | تأكد من نسخه بشكل صحيح |
| لا اتصال | تحقق من الإنترنت وأعد التحميل |
| الملف لم يُحفظ | انتظر ثانية وحاول مرة أخرى |
| الواجهة بطيئة | امسح الكاش وأعد التحميل |

## الأمثلة 📝

### صانع المحتوى
```
الموضوع: "الإنتاجية في العمل الحر"
الصيغة: "Inspirational"
النتيجة: تغريدة / سيناريو / منشور جاهز
```

### مساعد الدردشة
```
الرسالة: "هل تقبل المشروع بهذا السعر؟"
الأسلوب: "Diplomatic"
الرد: رد احترافي دبلوماسي
```

---

## نصائح احترافية 💡

1. **كن محدداً**: "تغريدة عن أهمية التعليم المستمر"
2. **جرب الأساليب**: نفس الموضوع بأساليب مختلفة
3. **احفظ المفضلات**: انسخ أفضل النتائج
4. **الجودة تحسن**: كلما أضفت تفاصيل أكثر = محتوى أفضل
